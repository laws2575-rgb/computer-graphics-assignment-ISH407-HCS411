 ABSTRACT

This report presents the implementation of a computer graphics project using OpenGL, focusing on rendering 2D and 3D objects using geometric primitives. The project demonstrates the OpenGL rendering pipeline, object construction, and animation techniques.

1. INTRODUCTION

OpenGL is a hardware-independent graphics API used for rendering 2D and 3D graphics. It provides a set of commands used to construct and manipulate graphical objects for interactive applications.

2. OPENGL RENDERING PIPELINE

The rendering process follows four main stages:

 2.1 Geometric Construction

Objects are created using primitives such as points, lines, triangles, and polygons.

2.2 Transformation & Viewing

Objects are positioned in 2D/3D space and viewed from a selected camera perspective.

 2.3 Color Application

Colors are assigned using direct specification, lighting, or texture mapping.

2.4 Rasterization

Mathematical representations are converted into pixels displayed on the screen.

 3. GEOMETRIC PRIMITIVES USED

* GL_POINTS – individual points
* GL_LINES – line segments
* GL_LINE_STRIP – connected lines
* GL_LINE_LOOP – closed loop
* GL_TRIANGLES – triangles
* GL_TRIANGLE_STRIP – connected triangles
* GL_TRIANGLE_FAN – fan-shaped triangles
* GL_QUADS – four-sided polygons
* GL_QUAD_STRIP – connected quadrilaterals
* GL_POLYGON – convex polygons
